# FIFO User Guide

## What FIFO means

FIFO rosters split your cycle into:

- **Work block**: consecutive days on-site
- **Rest block**: consecutive days at home/off

Examples: `8/6`, `7/7`, `14/14`, `14/7`.

## Setting up FIFO in Ellie

1. During onboarding, choose **FIFO / Swing Roster**.
2. Pick a preset FIFO pattern or **Custom FIFO**.
3. Set your current position using the FIFO phase selector:
   - Work block day X
   - Rest block day X
4. Set your cycle start date and shift times.

## Reading the dashboard

- FIFO calendar uses **Work Block** and **Rest Block** legend labels.
- The status card shows:
  - `WORK BLOCK` or `REST BLOCK`
  - `Day X of Y`
- Countdown text shows days until the next block.

## Voice examples

- “How many days until I’m home?”
- “When is my next work block?”
- “What block am I in right now?”
- “Am I working on March 10?”

## If voice doesn’t respond

1. Tap the mic manually (wake-word can be unavailable per session).
2. Check network connection.
3. Retry the request from idle state.
