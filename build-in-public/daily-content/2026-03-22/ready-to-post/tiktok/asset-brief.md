# TikTok Asset Brief

- Audience: shift workers, builders, and creator-native discovery viewers
- Objective: feel human and native enough to hold watch time and trigger replies
- Native format: creator-led vertical story with direct language
- Target length: 18-35 seconds
- Hook rule: Open with tension or confession, not setup.
- Structure: hook, what felt wrong, what changed, proof, and comment question
- Cover style: confession or surprising product truth
- CTA style: comment-led
- Overlay mode: light-builder
- Angle mix: Micro-Drama + User Empathy

## Story Fit

- Chapter: language pass across onboarding moments
- Hook: If users must decode roster wording, they leave before value appears.
- Proof line: Setup, Aha preview, and paywall now reinforce one promise: set once, preview upcoming shifts clearly.
- Builder overlay: Built from real mining-shift routines.

## Asset Needs

- spoken script
- caption
- cover text
- pinned comment
