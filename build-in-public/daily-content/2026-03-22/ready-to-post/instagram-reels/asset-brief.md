# Instagram Reels Asset Brief

- Audience: miners, shift workers, and broad discovery viewers
- Objective: earn a stop, a rewatch, and a comment inside a short vertical story
- Native format: short vertical video with fast visual clarity
- Target length: 12-25 seconds
- Hook rule: Start with a sharp line or visual contrast in the first beat.
- Structure: hook, real-life tension, proof moment, one lesson, and one question
- Cover style: short consequence or contrast line
- CTA style: question-led
- Overlay mode: light
- Angle mix: Micro-Drama + Before/After Reveal

## Story Fit

- Chapter: language pass across onboarding moments
- Hook: Readable copy can still fail if product promise is inconsistent.
- Proof line: Setup, Aha preview, and paywall now communicate one outcome in the same language.
- Builder overlay: Built from real mining-shift routines.

## Asset Needs

- on-screen opener
- voiceover
- caption
- cover text
- pinned comment
